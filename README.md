grunt-minify
============

GruntJS - minify css and js, compress images

Develop dir: /public

Production dir: /dist

1. Install npm modules
----------------------------------

    npm install
    
    
2. Run Grunt (minify CSS and JS, compress images)
----------------------------------
       
    grunt build       


What's inside?
---------------

  * [**GruntJS**][1] - GruntJS - The JavaScript Task Runner

  * [**NPM**][2] - Node Packaged Modules
                     

Enjoy!
